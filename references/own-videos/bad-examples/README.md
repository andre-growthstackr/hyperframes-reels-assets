# bad-examples/

**What is this?** The "don't do this" set — your own videos (or specific moments) that missed.
Curated failures, each labelled with *why* it fails.

**When to use it?** As a negative reference for the AI. A clear failure example steers away from a
mistake as strongly as a good example pulls toward the target — so bad examples matter as much as
best ones. When a new draft drifts toward one of these failure modes, this folder is the
correction.

**What feeling?** These *break* the target feeling — that's the point. The target remains: premium
· calm authority · clean · digital · dark-default · coral used once.

## Failure catalogue — the "don't do this" reference

Each of these is a recurring reason a GrowthStackr short fails. Tag each bad example with the
one(s) it demonstrates:

- **Caption too low** — subtitles stuck at the bottom of the frame (bottom-stuck subtitles). Keep
  captions in the safe zone, not glued to the bottom edge.
- **Too much zoom** — over-aggressive punch-ins / constant zooming that feels frantic instead of
  calm and premium.
- **Stillbild too long** — a static frame (stillbild) held far too long; the video stalls and
  loses motion energy.
- **Music too loud** — background music overpowering the voice; it should sit under, not compete.
- **Looks too corporate** — corporate-stock feel; generic, sterile, off-brand. The opposite of
  clean-premium.
- **Too little motion** — flat and lifeless; not enough of the talking head + UI + motion-graphics
  motion grammar that defines the style.
