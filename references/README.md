# references/ — Track 2 (things that teach the AI the style)

Track 2 is the *style-teaching* half of the video-DNA library. Nothing here is dropped straight
into a render — instead these resources teach the AI what "good" looks and feels like, so the
`shortform-video` pipeline reproduces GrowthStackr's video DNA instead of generic social slop.

See [`../LIBRARY.md`](../LIBRARY.md) for the labelling standard, the `notes.md` template, the
feeling vocabulary, and the Git LFS rule for heavy media.

## Subfolders

| Folder | What |
|---|---|
| [`reference-videos/`](reference-videos/) | 10–30 outside videos that nail the feeling, sorted by type (`ai-tool-breakdown/ claude-code-tutorial/ saas-demo/ case-study/ offer-video/ comment-for-link/ motion-graphic-only/`). Each: `video.mp4` + `notes.md` + `screenshots/`. |
| [`own-videos/`](own-videos/) | Your own work — `raw/ edited/ best-hooks/ best-ctas/ best-examples/ bad-examples/`. Bad examples matter as much as good ones. |
| [`caption-library/`](caption-library/) | Caption *language* you like, by beat (hook/metric/proof/cta/transition) + banned styles. |
| [`scripts/`](scripts/) | Recurring hooks, CTAs, proof lines, objection handling, offer phrasing. |
| [`storyboard-examples/`](storyboard-examples/) | Full storyboards for each video format. |
| [`motion-references/`](motion-references/) | Specific *motions* you like (logo reveals, metric cards, browser zooms, flares) with "what I like" notes. |
| [`brand-examples/`](brand-examples/) | Frozen frames showing *right* coral use and *right* dark-mode feel. |

## The labelling standard (non-negotiable)

**Every folder** has a `README.md` answering, in this order: **What is this? · When to use it? ·
What feeling?** **Every reference video** carries a sidecar `notes.md`. Copy the template from
[`../LIBRARY.md`](../LIBRARY.md) — do not invent your own shape.

The **feeling vocabulary** the whole library targets: premium, calm authority, clean, digital,
dark-default, coral used once. **Never:** meme, childish, MrBeast-loud, corporate-stock, rainbow
captions, bottom-stuck subtitles.
