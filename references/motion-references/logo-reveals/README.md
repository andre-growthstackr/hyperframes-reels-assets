# motion-references/logo-reveals/

**Built component:** [`../../../animations/logo-reveal-coral`](../../../animations/logo-reveal-coral)

## What is this?
Clips of **brand mark / wordmark reveals** you like — the moment a logo or wordmark resolves onto a
dark frame. The reference feel; the built `logo-reveal-coral` animation is the implementation.

## When to use it?
Pull when building or tuning a logo tag-in (hook stinger, CTA end-card, watermark). Match the built
animation's timing and coral moment to the clips here.

## What feeling?
Premium, calm, confident. One coral moment, dark ground. Never spinny, never a glint sweep.

### What I like (target bullets)
- Fast in, slow settle — mark arrives quick, eases to rest.
- No bounce, no overshoot on the mark itself.
- Coral fires once (a dot, an underline, one letter), never a coral wash.
- Small glow, not a lens flare; the mark stays crisp.
- Clean exit — hard cut or a single soft fade, no lingering.
