# scripts

**What is this?**
Recurring **spoken** language for GrowthStackr shorts — the lines the talking head actually says,
grouped by intent: hooks, CTAs, AI-tool breakdowns, tutorial phrasing, objection handling, proof
lines, offer phrasing. This is the *script* layer (what's spoken) as distinct from
`../caption-library/` (what's on the pill). Everything is in **Swedish** because the brand ships
Swedish shorts.

**When to use it?**
When writing or scripting a short. Pull the file for the intent you're on; each file groups lines
into blocks with a short "when to use" note so you can grab the right register without rewriting the
voice each time.

| File | Intent |
| --- | --- |
| `hooks.md` | Opening lines that stop the scroll |
| `ctas.md` | The closing ask |
| `ai-tool-breakdowns.md` | Walking through an AI tool / workflow |
| `tutorial-phrases.md` | Step-by-step "here's how" connective phrasing |
| `objection-handling.md` | Answering the doubt before it's spoken |
| `proof-phrases.md` | Before/after and denial-pivot proof lines |
| `offer-phrases.md` | Naming the offer / what you do, without pitching |

**What feeling?**
A senior operator talking. Direct, Swedish directness, denial-pivot (*det här är inte X, det är Y*),
outcome-first, full-stop emphasis, anti-hype. Premium and calm — never salesy, never MrBeast-loud,
never corporate-stock. The voice assumes the viewer is smart and busy.

## Voice rules (apply to every line here)

- **Outcome-first.** Lead with the result or the change, not the feature. *"Färdig reel ut"* before
  *"vi använder en pipeline"*.
- **Denial-pivot.** Define by what it *isn't*: *"Det här är ingen mall. Det är din råa video, klar."*
- **Full-stop emphasis.** Short sentences. The stop is the punch. *"Utan redigering."*
- **Swedish directness.** Plain, unhedged, a little blunt. No "kanske skulle man kunna".
- **Anti-hype, no banned words.** Never `revolutionera`, `unlock`, `supercharge`, `10x`, `seamless`,
  `plug-and-play`, `growth hacking`, `tillväxthack`, `game changer`. No em-dash.
- **Truth rule.** Any result claimed as real must be real; otherwise phrase it as a capability, not
  a figure.

## How lines pair with the pipeline

A spoken line and its caption are different lengths: the script line is a full sentence, the caption
is 1–3 word chunks pulled from it. When you use a line here, the matching caption beat lives in
`../caption-library/`. Example: script hook *"Det här är det snabbaste sättet att göra reels."* →
caption chunks `det här är` · `det **snabbaste**` · `sättet` (see `hook-captions.md`).
