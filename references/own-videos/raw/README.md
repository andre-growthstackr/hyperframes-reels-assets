# raw/

**What is this?** Unedited source footage and recordings from your own shoots — talking-head
takes, screen recordings, and whatever went in before the edit.

**When to use it?** As the before-half of the before/after pair with `../edited/`, so the AI can
learn your edit grammar. Also the pull-from pile when you want to re-cut an old recording.

**What feeling?** Raw is pre-feeling — but the material should be recorded to hit the target:
premium, calm authority, clean, dark-default. Discard takes that can never reach it.

Drop-in. Heavy media → **Git LFS** (see [`../../../LIBRARY.md`](../../../LIBRARY.md)).
