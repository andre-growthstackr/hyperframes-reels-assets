# b-roll/ — sparse, on-brand cutaway footage

**What is this?** A small, on-brand b-roll stock — typing, laptop-on-desk, screen close-ups, phone
scrolling, whiteboard, coffee-desk, recording setup, code on screen, dashboard review. Texture
shots to break up a beat.

**When to use it?** **Sparingly.** The GrowthStackr look is "talking head + UI + motion graphics,"
**not** generic stock b-roll. Reach for a b-roll clip only when a specific beat genuinely needs a
physical cutaway (e.g. a one-second hands-on-keyboard over a "just start typing" line) — never as
filler to pad runtime. If a UI screen or a motion graphic can carry the beat, use that instead.

**RULE: use sparingly.** Every b-roll insert should earn its place. Default to UI + motion; treat
b-roll as the exception, not the fallback.

**What feeling?** premium · calm authority · clean · digital · dark-default · coral used once.
Never corporate-stock — if a clip reads as generic stock footage, it doesn't belong here.
