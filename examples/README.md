# examples/

Whole prior video codebases, kept here so the skill (or you) can pull a specific effect,
timeline, or layout from a finished video and re-skin it for a new one.

Drop a complete HyperFrames project in its own folder (e.g. `examples/outbound-ai-45s/`)
with its `index.html`, `compositions/`, and assets. Reference a specific effect from it in a
beat ("use the counter from `outbound-ai-45s` scene-stat") — the skill copies that timeline
fragment and re-skins via `frame.md`.

Empty for now — add a project after your next render.
